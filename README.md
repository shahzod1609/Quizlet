# Quizlet
Quizlet App made Node js, Express js, MySql, React js.

## Installation
The quickest way to get start clone the project:
```Bash
$ git clone https://github.com/shahzod1609/Quizlet.git
```
Install dependencies:
```Bash
$ npm install
```
Start Express.js and React.js:

```Bash
$ npm start
```
